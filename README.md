# 🎤 Personal Voice Assistant

## 📌 Description
This is a Python-based Personal Voice Assistant that can understand voice commands and perform tasks like opening websites, telling time, and responding using speech.

## 🚀 Features
- Speech Recognition (voice input)
- Text-to-Speech (voice output)
- Open Google, YouTube, LinkedIn
- Tell time, date, and day
- Search queries on the web
- Open system applications like Calculator & Notepad

## 🛠️ Technologies Used
- Python
- SpeechRecognition
- pyttsx3

## ▶️ How to Run
1. Install required libraries:
   pip install SpeechRecognition pyttsx3 pyaudio

2. Run the program:
   python assistant.py

## 🎤 Sample Commands
- What is the time
- Open Google
- Search Python projects
- Tell me a joke
- Exit
