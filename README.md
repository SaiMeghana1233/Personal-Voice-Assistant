# Personal Voice Assistant 🎤

## Description
A Python-based voice assistant that can understand voice commands and perform tasks like opening websites, telling time, and searching online.

## Features
- Speech Recognition
- Text-to-Speech
- Open Google, Gmail, Calculator, Notepad
- Search queries on web

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3

## How to Run
1. Install libraries:
   pip install SpeechRecognition pyttsx3 pyaudio
2. Run:
   python assistant.py

## Sample Commands
- What is the time
- Open Google
- open calculator
- how are you
- who are you
- Exit
